STEPS FOR SETTING UP TESTING ENVIRONMENT:
- ALL NODES:
  1. $ sudo apt-get update
  2. $ git clone git@github.com:SudarsunKannan/butterflyeffect.git
  3. $ git checkout FullDynamic (run only installation part of install_cassandra.sh)
  4. $ sudo apt-get install -y openjdk-8-jdk maven ant
  5. $ sudo update-java-alternatives -s java-1.8.0-openjdk-amd64; java -version
  6. $ cd ~/butterflyeffect/code; source scripts/setvars.sh
  7. $ cd ~/butterflyeffect/code/mapkeeper/ycsb; rm -rf YCSB
  8. $ git clone https://github.com/brianfrankcooper/YCSB.git
  9. $ cd YCSB; mvn clean package
 10. $ cd ~/butterflyeffect/code/cassandra; bash setup.sh

- SERVER NODE:
  1. wget https://fossies.org/linux/misc/apache-cassandra-3.11.6-src.tar.gz/
  2. mv ~/apache.../pylib/cqlshlib path/to/python/site-packages
